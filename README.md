# FactoryScanner

FactoryScanner is a tool developed for scanning and managing assets in a factory environment. It provides functionality for scanning barcodes, QR codes, and other types of labels to track inventory and streamline processes within a factory setting.

## Features

- **Barcode Scanning**: Quickly scan barcodes and QR codes to retrieve information about assets.
- **Inventory Management**: Keep track of inventory levels and locations within the factory.
- **Customizable**: Easily customize the application to fit specific factory workflows and requirements.
- **Export Functionality**: Export scanned data for further analysis or integration with other systems.

## Installation

To use FactoryScanner, follow these steps:

1. Clone the repository:

    ```bash
    git clone https://github.com/szbalazs94/FactoryScanner.git
    ```

2. Navigate to the project directory:

    ```bash
    cd FactoryScanner
    ```

3. Install dependencies:

    ```bash
    npm install
    ```

4. Start the application:

    ```bash
    npm start
    ```

## Usage

1. Launch the FactoryScanner application.
2. Scan the barcode or QR code of the asset you want to manage.
3. View and update asset information as needed.
4. Export data for further analysis or integration with other systems.
